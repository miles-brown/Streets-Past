# Netlify Free Tier (Starter Plan) Research Report

## Research Summary
This report provides detailed information about Netlify's free tier (Starter plan) limits and features, compiled from their official pricing page.

## Netlify Free Tier (Starter Plan) Limits

### Bandwidth Limits
- **Bandwidth:** 100 GB per month
  - This is the total data transfer limit for all sites hosted on the free tier

### Build Minutes
- **Build Minutes:** 300 minutes per month
  - Includes both build and deploy time
  - Applies to all builds across all sites in the account

### Storage Limits
- **File Storage:** 300 MB per site
  - Maximum size for individual site files
  - Total storage across all sites in the account

### Function Invocation Limits
- **Serverless Function Invocations:** 300,000 per month
  - Includes all serverless functions deployed to Netlify
  - Shared across all functions in the account

### Edge Function Limits
- **Edge Function Invocations:** Limited based on overall usage
  - Specific limits not clearly specified in the pricing table
  - Subject to fair use policy

### Additional Features Included in Free Plan

#### Hosting & Deployment
- ✅ Unlimited personal repositories
- ✅ Branch previews (deploy previews)
- ✅ Custom domains (with Netlify subdomain included)
- ✅ SSL certificates (automatic)
- ✅ Form handling (100 submissions/month)
- ✅ Split testing
- ✅ Identity (user management) - Limited
- ✅ Analytics - Basic

#### Build & Development
- ✅ Custom build commands
- ✅ Environment variables
- ✅ Webhooks
- ✅ Site-wide redirects
- ✅ Header management
- ✅ Partial deploys
- ✅ Deploy notifications

#### Security Features
- ✅ Secrets controller
- ✅ Basic security features
- ✅ Visitor access control - Limited
- ✅ Environment variable security

#### Support
- ✅ Email technical support

### Limitations & Restrictions
- ❌ No priority support
- ❌ No team access control
- ❌ No private repositories (GitHub organizations)
- ❌ Limited Identity features
- ❌ No advanced security features
- ❌ No SOC 2 compliance
- ❌ Limited to 100 active sites (typical limit)
